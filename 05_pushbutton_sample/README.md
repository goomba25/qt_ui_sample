# QPushButton

일반적인 버튼 위젯.

## 자주 사용하는 메서드

* setText : 텍스트 입력
* setSizePolicy(QSizePolicy) : 사이즈 정책 변경
* setFixedSize(QSize) : 버튼 위젯의 사이즈를 해당 사이즈로 고정
* setCheckable(bool) : 체크가 가능한 버튼으로 만들지 결정
* setEnabled(bool) : 버튼의 사용 가능성 결정
* setChecked(bool) : 버튼의 입력상태 결정
